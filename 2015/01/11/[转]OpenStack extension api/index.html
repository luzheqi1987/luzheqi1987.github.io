因为上面是一个生成器，可以这样调用：
[python]self.controller.create(req, body).next()[/python]
或者用for循环调用。另外在单元测试时使用assertRaises时调用它时传给Mock的应该是一个函数地址，所以下面的调用在next后面不要加括号哦。
